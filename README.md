# nascalay-frontend

## For developer

### Run dev-server

```sh
$ yarn
$ yarn gen-api
$ yarn dev
```

### Lint
```sh
$ yarn lint
```

with fix
```sh
$ yarn fix
```

### Environment
Requires java
```sh
$ yarn gen-api
```
## Production

### Build

```sh
$ yarn build
```

### Start
```sh
$ yarn start
```
